For my final project in ASTR 1510, I decided to create a website about Albert Einstein. I used HTML to structure the content, CSS for styling, and JavaScript for interactivity. This combination allowed me to present information about Einstein in an engaging and interactive manner. HTML organized the content, CSS enhanced the visuals, and JavaScript added interactive elements, making my project informative and visually appealing.

Eldoria

This was my final project for the class CS 2013 - Programming with Data Structures. It's a text-based adventure game, telling the story of the medival town of Eldoria, that is haunted by an evil demon. The game was create with Java. My role was coding of the game, while I had support in writing the dialogue.
I coded a custom data structure for his project that would allow me to easily change and edit dialogue nodes in the future while being able to have branching storylines and different endings to the story.
