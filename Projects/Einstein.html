Albert Einstein

Project Overview

This project was created, as my final psroject for my class, ASTR 1510 in Fall 2022. The goal was to present a topic, in a way chosen ourselves. I decided to create a website about Albert Einstein, presenting different categories, such as some famous theories of him and his life.

Technologies Used

  • HTML
  • CSS
  • JavaScript
  • Adobe XD
  • GitHub Pages

Development Process

1. Plan

The first step for this process was to create a webpage structure, I decided which pages I need, and how they would be connected. In this case, I decided to create 2 main categories, general information about Einstein's life and another one with some important theories. I decided to introduce his 3 most famous theories, theory of relativity, quantum physics, and photoelectric effect. I also created a separate page for the sources I used.

2. Design

I designed the website to have a nostalgic, 90s website or game feeling. That is a pit playful and has the feel of a game menu, to make the relatively boring subject matter more interesting.

3. Development

For the design process, I started by creating the homepages HTML and CSS. After most of that was done, I applied the design to the other webpages, and adjusted where it was necessary.

4. Testing and Revision

After the website was functional, it was a little bit to boring for what I wanted to achieve, so to give the website an even more gamified look, I decided to also implement a cursor, and depending on the browser custom scrollbar.

5. Launch

I wanted to make the project accessible to my professor, so I decided to publish it with GitHub pages
